# XQuant

Implementation of [EMNLP 2025] [XQuant: Achieving Ultra-Low Bit KV Cache Quantization with Cross-Layer Compression](https://arxiv.org/abs/2510.11236)

## Overview

XQuant is a plug-and-play framework for ultra-low-bit KV cache quantization with data-free calibration and cross-layer compression, achieving efficient compression without sacrificing performance.
